# Super Hero service

* .Net7 Blazor Web Assembly

## Page 
** /SuperHeroes
* 查詢所有super hero
** /SuperHero
* 新增super hero
* /SuperHero/{id}
* 修改super hero
* 刪除super hero

## API 
** 後端api https://github.com/m9590207/DotNet7MinimalAPI
** 查詢Super Hero by ID 
* GET /api/SuperHero/{id}
** 更新Super Hero by ID
* PUT /api/SuperHero/{id}
** 刪除Super Hero by ID
*DELETE /api/SuperHero/{id}  
** 新增Super Hero 
* POST /api/SuperHero
** 查詢所有Super Hero
* GET /api/SuperHero
